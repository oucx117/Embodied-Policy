# 扩散模型

### 基本定义

扩散模型是一种生成模型，尤其擅长创造高质量和多样性的图像。其核心思想非常直观，分为两个过程：

* 前向加噪：输入清晰图像，逐步加入高斯噪声，直到得到纯粹的随机噪声图；
* 逆向去噪：输入随机噪声图，逐步去除图像中的噪声，直到得到清晰的图像。

### 前向加噪过程

* 机制：马尔科夫链，第 `t` 个时间步的图像 `x_t` 只和前一个时间步 `x_{t-1}` 有关；

* 超参数：

  * `\alpha_t\,(1\leq t \leq T)`：控制每个时间步加入噪声的强度大小；
  * `\epsilon`：标准高斯噪声，全时间步共享；

* 加噪公式：
  ``
  x_t = \sqrt{\alpha_t}\cdot x_{t-1} + \sqrt{1-\alpha_t}\cdot \epsilon
  ``

​	实际加噪时，并不是按顺序一步一步迭代的，而是一步到位：`x_t=\sqrt{\overline{\alpha}_t}\cdot x_0+\sqrt{1-\overline{\alpha}_t}\cdot \epsilon`

### 逆向去噪过程

* 机制：根本任务是从 `x_t` 还原出 `x_{t-1}`，但是模型并不是直接从 `x_t` 预测出 `x_{t-1}`，而是先通过当前信息（包括 `x_t` 和 `t`）预测出加入到 `x_t` 中的噪声 `\epsilon`，再从 `x_t` 中“减去”这个噪声，从而还原出 `x_{t-1}`； 

* 预测噪声（监督学习）：

  * 模型：U-Net 神经网络 `\epsilon_\theta`
  * 输入：`x_t` 和 `t`
  * 输出：预测噪声 `\epsilon_\theta(x_t,t)`
  * 计算预测噪声 `\epsilon_\theta(x_t,t)` 和真实噪声 `\epsilon` 之间的误差，反向传播更新模型

* 去除噪声：

  * 核心挑战：在生成图片时，我们只有模型的预测噪声 `\epsilon_\theta(x_t,t)`，而没有真实噪声 `\epsilon_\theta(x_t,t)`，所以无法简单地套用前向公式的代数逆运算，否则会得到极不稳定的生成结果。

  * “理想推导，现实逼近”策略：

    * 目标：我们可以通过求解后验分布 `p(x_{t-1}|x_t)` 的均值，用均值作为输出，则会得到更稳定的还原结果；

    * 理想推导：直接求解后验分布 `p(x_{t-1}|x_t)` 十分困难，因此我们转而求解一个在数学上可行的“理想”问题：在已知 `x_0` （初始图）的情况下，求解 `p(x_{t-1}|x_t, x_0)`；由于引入了 `x_0` 这个锚点，并利用马尔可夫性质，这个问题可以被分解为一堆已知的高斯分布，从而推导出一个精确的均值公式（其中 `\overline{\alpha}_t = \prod_{i=1}^t\alpha_i`）：
      ``
      \mu_t(x_t,x_0) = \frac{\sqrt{\overline{\alpha}_{t-1}}(1-\alpha_t)}{1-\overline{\alpha}_t}x_0+\frac{\sqrt{\alpha_t}(1-\overline{\alpha}_{t-1})}{1-\overline{\alpha}_t}x_t
      ``

    * 现实逼近：为了得到 `p(x_{t-1}|x_t)`，我们需要消去均值公式中的 `x_0` 项，考虑到
      ``
      x_t=\sqrt{\overline{\alpha}_t}\cdot x_0+\sqrt{1-\overline{\alpha}_t}\cdot \epsilon
      ``
      上式可通过嵌套加噪公式，并结合 `N(0,\sigma_1^2)+N(0,\sigma_2^2)=N(0,\sigma_1^2+\sigma_2^2)` 求得；由于真实的噪声 `\epsilon` 在去噪过程并不知道，所以我们用预测噪声 `\epsilon_\theta(x_t,t)` 近似；通过联合上面两式消去 `x_0`，可以得到：
      ``
      \mu_t \approx \frac{1}{\sqrt{\alpha_t}}\left(x_t - \frac{1-\alpha_t}{\sqrt{1-\overline{\alpha}_t}}\epsilon_\theta(x_t,t)\right)
      ``
      这个均值本质上是对我们真正想求的 `p(x_{t-1}|x_t)` 的一个高质量近似解。

    * 去噪公式：
      ``
      x_{t-1}=\frac{1}{\sqrt{\alpha_t}}\left(x_t - \frac{1-\alpha_t}{\sqrt{1-\overline{\alpha}_t}}\epsilon_\theta(x_t,t)\right)+\sqrt{1-\alpha_t}z
      ``
      其中前半部分对应刚刚所求的均值，表示还原后的结果，而后半部分是一个微小噪声，用于增加生成的多样性。

### 文生图

一般的扩散模型只能生成和原训练数据风格类似的图片，而“文生图”指的是引入文本提示，让模型生成符合文本描述的图片，这需要引入以下两个关键组件：

* **文本编码器**
  * 挑战：U-Net 只能理解数字，无法理解人类的语言文字；
  * 解决：使用文本编码器，将输入的文本提示（Prompt）转换成一个包含丰富语义信息的数字向量（Embedding）；
  * 常用工具：CLIP 的文本编码器是目前最主流的选择。它通过对海量“图片-文本”数据对的学习，能够深刻理解文本的含义，并将其映射到一个特殊的“语义空间”中。
* **交叉注意力机制**
  * 挑战：U-Net 在埋头预测噪声时，如何才能有效地“参考”我们给出的文本指令向量？
  * 解决方案：在 U-Net 的内部结构中，加入交叉注意力模块；
  * 工作方式：通过交叉注意力计算每个图片区域对每个词 token 的注意力，从而让模型知道当前区域应该更关注哪些文本信息
  * 结果：U-Net 不再是简单地预测噪声，而是预测一个在文本指令引导下的、带有明确目的性的噪声。

#### 训练过程

* 数据源：使用大规模的（图片，文本描述）数据对进行训练，例如 LAION 数据集；

* 模型：带有交叉注意力模块的 U-Net 神经网络 `\epsilon_\theta`
* 输入：加噪图像 `x_t` / 时间步 `t` / 文本向量 `c`
* 输出：预测噪声 `\epsilon_\theta(x_t,t,c)`
* 计算预测噪声 `\epsilon_\theta(x_t,t,c)` 和真实噪声 `\epsilon` 之间的误差，反向传播更新模型

#### 推理流程

1. 用户输入：一个文本提示
2. 文本编码：文本编码器将该提示转化为一个指令向量 `c`
3. 初始化：生成一张与目标输出尺寸相同的纯随机噪声图 `x_T`。
4. 引导式去噪循环（从 `t=T` 到 `1`）：
   - 在每一步 `t`，U-Net 都会接收当前的噪声图 、时间步 `t` 和指令向量 `c`。
   - U-Net 在 `c` 的引导下（通过交叉注意力），预测出当前最应该被移除的噪声 `\epsilon_\theta`。
   - 从 `x_t` 中减去这个被引导的噪声，得到一个更清晰且更符合文本描述的图像 `x_{t−1}`。
5. 最终呈现：当循环结束，一张符合文本提示的清晰图像就诞生了。
